🧭 What has Wiped?

  • This is a map wipe only — blueprints, XP, and Prestige remain.
  • Major focus on new questlines — grind them early for maximum rewards!
  • Boss fights, loot tables, and economy systems have been completely overhauled.

🧾 General Notes

  • This is a map wipe only.
  • Blueprints, XP, and Prestige remain untouched.
  • Main focus: New questlines — race to complete them first!

📜 NEW QUESTLINES

🎯 Major Addition: Two brand new questlines to conquer this wipe!

  • Quartermaster – Sulfur
    • 5 Tiers (T1-T5)
    • Progressive sulfur-based challenges
    • Unlock powerful rewards as you advance
  • The Boom God Questline
    • All-new explosive-themed quest series
    • Test your raiding and explosive skills
    • Exclusive rewards for completion
Pro Tip: Start these questlines early — first completions get bragging rights!

🐉 Boss & Events

  • Boss Fight
    • Completely reworked mechanics
    • Improved difficulty scaling
    • Better loot distribution
  • Legendary Heli
    • Loot table rebalanced
    • Reduced battery drops (fewer batteries = better variety)
    • Improved reward quality

💰 Loot & Economy

  • RP (Reward Points)
    • Stack size increased: 10,000 → 1,000,000
    • RP Coins now drop in loot tables
    • Easier to manage large amounts of currency
  • Blueprint Fragments
    • New item added to progression system
    • Found in various loot sources
  • Casino Changes
    • Scrap Box removed from Casino perk
    • Better reward balancing
  • Boss Loot
    • Complete rework of drop tables
    • More balanced and rewarding

⚙️ Stability & Performance

  • Sync Plugins Reworked
    • Smoother cross-server node transfers
    • Reduced lag during world hopping
    • Improved data synchronization
  • Backend optimizations for better server performance
  • Enhanced stability across all systems

🏆 The Challenge

This wipe is all about the NEW QUESTLINES — who's conquering them first?

Race to complete the Quartermaster and Boom God questlines. Show off your progress in Discord and claim your bragging rights! 🎯

🎮 Good Luck, Survivors!

Get out there and start grinding those new questlines. 💪
